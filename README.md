# ✂️ BG Remover

Бесплатный инструмент для удаления фона с фотографий. Работает прямо в браузере — без регистрации, без водяных знаков и без загрузки ваших фото на какие-либо серверы.

## Как пользоваться

1. Откройте сайт.
2. Перетащите фото на страницу, нажмите «Выбрать файл» или просто вставьте картинку из буфера обмена (Ctrl+V).
3. Подождите несколько секунд — нейросеть сама отделит объект от фона.
4. Сравните результат «до / после» и скачайте картинку с прозрачным фоном в PNG или WEBP.

Поддерживаются форматы PNG, JPG, WEBP и AVIF, размер файла — до 30 МБ.

## Выбор модели

Над зоной загрузки есть выпадающий список с двумя режимами работы:

| Модель | Размер загрузки | Кому подойдёт |
| --- | --- | --- |
| ⚡ Лёгкая | ~42 МБ | Медленный интернет и обычные фото — качество почти не отличается |
| 💎 Нормальная | ~84 МБ | Сложные кадры: волосы, мех, мелкие детали |

Ваш выбор сохраняется — при следующем визите ничего настраивать не нужно.

## Почему первый запуск дольше обычного

При первом использовании браузер скачивает модель нейросети (42–84 МБ в зависимости от выбранного режима). Это происходит только один раз: дальше модель берётся из кеша, и обработка запускается практически мгновенно даже при медленном интернете.

## Приватность

Вся обработка происходит локально на вашем устройстве:

- Фотографии **не загружаются ни на какие серверы** и не покидают ваш браузер
- Сайт не требует регистрации и не собирает личные данные
- Можно спокойно обрабатывать личные и рабочие фото

## Частые вопросы

**Это бесплатно?**
Да, полностью. Без лимитов, подписок и водяных знаков.

**Какое качество на выходе?**
Разрешение исходного фото сохраняется, фон становится прозрачным.

**Почему ничего не происходит?**
Для первого запуска нужно интернет-соединение, чтобы скачать модель. Если загрузка оборвалась — просто попробуйте ещё раз.

**На телефоне работает?**
Да, в любом современном мобильном браузере. На слабых устройствах обработка может занять чуть больше времени.
